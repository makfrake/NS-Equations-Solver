# 12-Steps Navier-Stokes Equations Solver
The solver is based on the famous CFD course taught by Prof. **Lorena A. Barba** as a module of **Computational Fluid Dyanamics** class at Boston University.  
You can find the project at the following link: [12 steps to Navier-Stokes](https://lorenabarba.com/blog/cfd-python-12-steps-to-navier-stokes/)
